>「細かいところだけどルールの投票について～」の質問-- 2020-09-11 (金) 18:04:07 ------------ まず投票項目に「どちらでもいい」を追加した背景には、管理者の「全体の空気感（多数派がなんであるか）を大事にしたい」という想いが根底にあります。 「はい」、「いいえ」だけだと議論に興味のない利用者(データやAARだけ見て、コメントやAARを書かない方たちが多いと思います)が無視されるのではないか、 という危惧があったため、そのような閲覧のみの利用者の方々にも投票しやすい選択肢を設けたかった、というのが「どちらでもいい」を項目に入れた理由です。 次に「『どちらでもいい』の項目は『どちらかといえば賛成』と『どちらかといえば反対』に分けた方がいい」というご提案についてですが、 管理者・Discord議論参加者ともに「閲覧のみの利用者にとっても、『消極的だけど意思決定を促す選択肢』が増える」と前向きの受け取り方でした。 詳しくは[[wiki暫定ルール置き場]]をご覧いただきたいのですが、従来の3項目(賛成/反対/どちらでもいい)に代えて (賛成/どちらかといえば賛成/どちらともいえない/どちらかといえば反対/反対)の5項目の投票先を設けることとします。 また厳密には「どちらでもいい」を選択肢から無くすこともご提案には含まれておりましたが、これに関しては 「白票であることを理解した上で『どちらともいえない』に投票した人」の割合も重要な指標となるため、選択肢として残しました。 ------------ >凍結時に外部じゃなくてwiki内部に議論用の避難所コメ欄作らないのなぜ？ -- 2020-09-11 (金) 21:13:33 ------------ 9/10の回答にもある通り、場所をwiki外に移すことで空気の入れ替えを期待しているためです。 ------------ >参加手続きやルールの類はどうなってるの？英語ユーザーも読むから英語に翻訳しにくい単語はダメとかあったり？ -- 2020-09-12 (土) 19:15:10 ------------ まずDiscordでの議論の参加に手続きなどはなく、Discordの登録を行えばページ内のDiscordへのリンクから議論用チャンネルへの参加が可能です。 次に今回投票に掛けるルールは[[wiki暫定ルール置き場]]にあるものが全てです。英語版ユーザーの利便性も考慮して、ゲーム内で使用される用語の英和併記をお願いしているものであり、あくまでお願いであってルールではありません。 「細かいところだけどルールの投票について～」の質問への回答を踏まえて、ルール案下部の投票に関する部分を修正する必要があったので、その修正案です。 以下の文面に入れ替えることを想定しています。 ------------ 以上が新たなルール案です。このうち「2. ルール」に含まれる4項目から「2.4 その他」を除く3項目について(賛成/どちらかといえば賛成/どちらともいえない/どちらかといえば反対/反対)のいずれかに投票して戴く予定です。 「賛成」と「どちらかというと賛成」に投じられた票が「どちらかといえば反対」と「反対」を上回った項目を可決とします。((「賛成」と「どちらかといえば賛成」の間で票の重みづけ(前者を2点後者を1点など)は行いません。1票は1票です。逆も同様です。)) 「どちらともいえない」は白票として扱います。ただし、たとえルール案の可否に関与しない点が棄権(投票しない)と同じであっても、 「''白票であると理解した上で『どちらともいえない』に投票した人''」 の割合も重要な指標となりますので、白票という意思表示につながる投票への参加をお願いします。 2020/9/8の開示から1週間程度、ルール案の周知徹底と疑問点の解消のための質疑応答期間を設けます。 質問は[[コメントコーナー/wiki運営議論用]]又は[[Disocrd>https://discord.com/invite/EyYWxcZ]]にお寄せください。 頂いた質問とそれに対する回答は、コメントコーナー/wiki運営議論用に掲載予定です。 同様の質問が既に回答済みである場合もありますので、予めそちらに目を通してから質問して頂けると無駄がないかと思われます。 ------------ (edited)
